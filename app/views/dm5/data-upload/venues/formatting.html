Venue file format requirements

Your data upload file must be a CSV file type. If you have more than one option when saving as a CSV, choose,
CSV UTF-8 (Comma delimited) (.csv).

The tables below show information about each field in the data upload file and how to complete it correctly.

  1. Your venue reference
  2. Venue name
  3. Address line 1
  4. Address line 2
  5. Town or City
  1. County
  2. Postcode
  3. Email
  4. Phone
  5. Website

YOUR_VENUE_REFERENCE

Mandatory
Yes
Character limit
255
Notes
You must include a venue reference on your data upload.
A venue reference must not be edited or removed on a data upload.

VENUE_NAME

Mandatory
Yes
Character limit
250
Notes
A venue name must not be duplicated.
This information will be visible when linked with the courses you provide.
