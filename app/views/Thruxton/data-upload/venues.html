You can upload a CSV file to update multiple venues in Publish to the course directory. If you are completing a venue data upload for the first time we recommend you take the following steps:

Download your published venue data
This will ensure all your existing venues and venue references will be included in the data file.

Add new venues to your file
Follow the file format requirements for each of the fields.

Add a venue reference
An existing venue reference should not be edited or removed.

Removing a venue from a data upload
You can remove a venue from your data upload by deleting all data within the row.

You should not remove a venue from your data file that is linked to a course. Removing a venue from your data file that is linked to a course will be added when publishing to Publish to the course directory.

Edit or delete a venue that is linked to a course
You will need to edit or delete the the courses by selecting view and edit from the homescreen
