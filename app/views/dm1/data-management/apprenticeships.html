Preparing your apprenticeship data

You can upload a CSV file to update your apprenticeship training courses in Publish to the course directory.

Creating your data file

To create your file you can download:

  • an apprenticeship data template (CSV)

Completing your data file

To ensure an upload is successful you must:

  • add new venues to the service prior to an upload
  • include existing apprenticeships and new apprenticeships in the file
  • complete all mandatory fields
  • follow the format requirements
  • save the file as a CSV
