How to prepare a bulk upload file

You can upload a CSV file to update your courses, or you can add, edit or delete each course individually.

Creating your bulk upload file

Uploading and publishing a bulk upload file will replace all of your published courses. To create the file you can:

  • download a blank template and add your course information
  • download and edit a file containing your published course information
  • get help to fill out your bulk upload file
